# Guild Wars 2 - Raid LiveSplit Autosplitter

This tool connects Guild Wars 2 (via the Mumble Link) with LiveSplit, allowing automatic start, reset, and splits based on in-game events.

## Installation Steps

1. [Download](https://github.com/Meriksen95/ECHO-Raid-Pack/releases/latest/download/ECHO-Raid-Pack.taco) the files:
   - LiveSplit.GW2.dll
   - gw2_splits.json
   - GW2 - Raid Full Clear.lss
   - GW2 - Autosplitter.lsl

2. Install the autosplitter files

Place the files in the following locations inside your **LiveSplit installation folder**:
> LiveSplit/Components/**LiveSplit.GW2.dll**
>
> Livesplit/Components/GW2Splitter/**gw2_splits.json**

Note: you create the *GW2Splitter* folder yourself



4. Load the splits and layout.

Open LiveSplit, right click the window choose 'open splits' and choose from file. This file contains your times.
Do the same with 'layout'. This file is the visual file and the autosplitter control.
